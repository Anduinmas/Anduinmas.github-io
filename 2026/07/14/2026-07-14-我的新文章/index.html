你好，世界！我的博客上线啦
📝 写在前面：花了几个晚上，终于把属于自己的一隅小天地搭建好了。想在这里记录一些东西——技术的、生活的、读书的、瞎想的。写下来，才算真正拥有过。 为什么要写博客？这个问题我问过自己很多次。 市面上有掘金、知乎、微信公众号，似乎不缺一个发声的地方。但每次想在那些平台写点什么，总有一股无形的不自在——排版要迎合流量、标题要追求点击、内容要追逐热点。 我只想安安静静地写点东西。 所以有了「一隅小记」。一个完全属于我的角落。不用考虑 SEO 标题怎么写，不用纠结封面图吸不吸引人。写技术就写技术，写生活就写生活，写得长就长，短就短。 怎么搭起来的？对于一个只会简单前端的人来说，搭博客其实比想象中简单。 技术栈：Hexo + Butterfly 主题 + GitHub Pages，完全免费。 流程大概是这样的： 装好 Node.js 和 Git（这两个是基础） 用 hexo init 初始化项目 选一个好看的主题——Butterfly，一眼就爱上了它的文艺气质 配置一堆功能：搜索、暗黑模式、RSS、评论、访问统计 部署到 GitHub Pages，绑定域名 开始写第一篇文章 ...
